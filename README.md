# my-vue2-app
用于学习 vue 知识，集成vuex 进行组件状态管理，vue-router 控制页面间跳转




## Project setup
```
yarn install
```
## 上一步如果报engine检查的错误请执行
```
yarn config set ignore-engines true
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
